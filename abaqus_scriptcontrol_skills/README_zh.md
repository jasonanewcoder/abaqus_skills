# Abaqus 技能语料库 (Abaqus Skills Corpus)

## 项目简介

本技能库旨在为 AI 编程助手提供结构化的 Abaqus 二次开发知识，帮助用户通过自然语言描述快速生成高质量、可运行的 Abaqus Python 脚本。

## 核心目标

- **降低门槛**：用户无需记忆复杂的 Abaqus API，通过自然语言即可生成脚本
- **保证质量**：提供经过验证的代码模板和最佳实践
- **覆盖全面**：从基础建模到高级分析（静力/疲劳/XFEM/热应力/复合材料）
- **快速迭代**：支持对话式需求澄清，快速收敛到用户真实需求

## 技能模块说明

### 通用技能 (general/)

所有分析类型的基础操作：
- 几何建模（草图、拉伸、旋转、切割）
- 材料属性定义（各向同性/各向异性）
- 截面属性（实体/壳体/梁）
- 装配与约束
- 分析步设置
- 边界条件与载荷
- 网格划分技术
- 作业提交与监控
- 结果提取与可视化

### 静力分析 (static/)

- **线性静力分析**：小变形、线弹性材料
- **非线性静力分析**：大变形、材料非线性、接触

### 疲劳分析 (fatigue/)

- **高周疲劳**：基于应力的 S-N 曲线方法

### 扩展有限元 XFEM (xfem/)

- **裂纹分析**：扩展有限元方法、损伤演化

### 热应力分析 (thermal/)

- **稳态热传导**：温度场计算
- **瞬态热传导**：随时间变化的温度场
- **热-力耦合**：热应力计算

### 复合材料 (composite/)

- **层合壳单元**：经典层合板理论实现
- **失效分析**：Hashin 失效准则

## Claude Code Skill 格式说明

本技能库按照 Claude Code 的 Skill 格式组织：

- **SKILL.md**：技能的主文件，包含技能的概述、功能列表、代码片段和快速参考
- **reference/**：详细参考文档目录，包含完整的 API 参考和代码模板

### 使用方式

1. **查看 SKILL.md**：了解技能的功能和基本用法
2. **查看 reference/**：获取详细的代码模板和 API 参考
3. **组合使用**：根据需求组合多个技能模块生成完整脚本

## 快速开始

### 给用户的使用流程

1. **描述需求**：用自然语言描述你的分析需求
2. **AI 澄清**：AI 会通过简洁的对话确认关键细节
3. **生成脚本**：AI 根据确认的需求生成完整脚本
4. **运行验证**：在 Abaqus 中运行脚本，查看结果

### 给 AI 的使用指南

AI 助手在生成脚本前应遵循以下原则：

1. **先理解，后生成**：通过 2-3 个关键问题确认用户意图
2. **模块化构建**：使用本库的技能模块组合成完整脚本
3. **提供注释**：代码中包含清晰的步骤说明
4. **包含验证**：生成脚本后提供验证清单

详见 [`prompts/ai_guide.md`](prompts/ai_guide.md)
