# Abaqus 技能语料库 (Abaqus Skills Corpus)

## 📋 项目简介

本技能库旨在为 AI 编程助手（如 Vibe Coding、OpenClaw 等）提供结构化的 Abaqus 二次开发知识，帮助用户通过自然语言描述快速生成高质量、可运行的 Abaqus Python 脚本。

## 🎯 核心目标

- **降低门槛**：用户无需记忆复杂的 Abaqus API，通过自然语言即可生成脚本
- **保证质量**：提供经过验证的代码模板和最佳实践
- **覆盖全面**：从基础建模到高级分析（静力/疲劳/XFEM/热应力/复合材料）
- **快速迭代**：支持对话式需求澄清，快速收敛到用户真实需求

## 🚀 快速开始

### 给用户的使用流程

1. **描述需求**：用自然语言描述你的分析需求
   > "我需要分析一个承受内压的圆筒，材料是钢，壁厚 10mm，直径 200mm"

2. **AI 澄清**：AI 会通过简洁的对话确认关键细节
   > "确认几个问题：1) 内压大小？2) 是否需要考虑端盖效应？3) 是线弹性分析吗？"

3. **生成脚本**：AI 根据确认的需求生成功率完整的 Python 脚本

4. **运行验证**：在 Abaqus 中运行脚本，查看结果

### 给 AI 的使用指南

AI 助手在生成脚本前应遵循以下原则：

1. **先理解，后生成**：通过 2-3 个关键问题确认用户意图
2. **模块化构建**：使用本库的技能模块组合成完整脚本
3. **提供注释**：代码中包含清晰的步骤说明
4. **包含验证**：生成脚本后提供验证清单

详见 [`prompts/ai_guide.md`](prompts/ai_guide.md)

## 📚 技能模块说明

### 通用技能 (general/)

适用于所有分析类型的基础技能：
- 几何建模（草图、拉伸、旋转、切割）
- 材料属性定义（各向同性/各向异性）
- 截面属性（实体/壳体/梁）
- 装配与约束
- 分析步设置
- 边界条件与载荷
- 网格划分技术
- 作业提交与监控
- 结果提取与可视化

### 静力分析 (static/)

- **线性静力分析**：小变形、线弹性材料
- **非线性静力分析**：大变形、材料非线性、接触
- **屈曲分析**：特征值屈曲、后屈曲

### 疲劳分析 (fatigue/)

- **低周疲劳**：基于应变的疲劳寿命预测
- **高周疲劳**：基于应力的 S-N 曲线方法
- **疲劳裂纹扩展**：Paris 定律应用

### 扩展有限元 XFEM (xfem/)

- **静态裂纹**：固定裂纹的应力强度因子计算
- **动态裂纹扩展**：基于损伤准则的裂纹扩展模拟
- **内聚力模型**：粘结界面开裂分析

### 热应力分析 (thermal/)

- **稳态热传导**：温度场计算
- **瞬态热传导**：随时间变化的温度场
- **热-力耦合**：热应力计算
- **顺序耦合热应力**：先热后力的分析流程

### 复合材料 (composite/)

- **层合壳单元**：经典层合板理论实现
- **实体复合材料**：3D 复合材料建模
- **失效分析**：Hashin、Puck、LaRC 等失效准则
- **渐进损伤**：材料退化和刚度折减
