# Abaqus DISP 自定义边界条件子程序技能

## 技能描述

本技能指导AI生成Abaqus Standard的自定义位移边界条件子程序DISP，实现复杂位移控制，如正弦振动、多点运动协调等。

## 适用场景

- 正弦/随机振动激励
- 多自由度协调运动
- 位移控制的疲劳试验模拟
- 几何非线性大位移问题

## 关键特性

| 特性 | 说明 |
|------|------|
| 分析类型 | 隐式/显式动力学 |
| 控制类型 | 位移、速度、加速度 |
| 时间函数 | 任意时间相关函数 |
| 空间分布 | 基于坐标、节点集的分布 |

## DISP 接口定义

```fortran
SUBROUTINE DISP(U,KSTEP,KINC,TIME,NOEL,NPT,COORDS,JLTYP,
     1 SNAME)
```

## 关键参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| U(1) | 输出 | 位移1（X方向或自由度1）|
| U(2) | 输出 | 位移2（Y方向或自由度2）|
| U(3) | 输出 | 位移3（Z方向或自由度3）|
| TIME(1) | 输入 | 当前增量步时间 |
| TIME(2) | 输入 | 总时间 |
| COORDS(3) | 输入 | 节点坐标 |
| NOEL | 输入 | 单元编号（节点编号）|
| KSTEP | 输入 | 分析步编号 |

## 位移控制类型

### 1. 正弦振动位移

```fortran
      SUBROUTINE DISP(U,KSTEP,KINC,TIME,NOEL,NPT,COORDS,JLTYP,
     1 SNAME)
C
C  正弦振动位移边界条件
C  幅值和频率可调
C
      INCLUDE 'ABA_PARAM.INC'
C
      DIMENSION U(3), TIME(2), COORDS(3)
      CHARACTER*80 SNAME
C
      REAL*8 AMP, FREQ, PHASE, T, PI
      PARAMETER(PI=3.141592653589793D0)
C
C-----------------------------------------------------------------------
C  振动参数
C-----------------------------------------------------------------------
      AMP   = 1.0D-3       ! 振幅 (m)
      FREQ  = 10.0D0       ! 频率 (Hz)
      PHASE = 0.0D0        ! 初始相位 (rad)
C
C-----------------------------------------------------------------------
C  计算位移
C-----------------------------------------------------------------------
      T = TIME(2)
C
C  仅X方向施加正弦位移
      U(1) = AMP * SIN(2.0D0*PI*FREQ*T + PHASE)
      U(2) = 0.0D0
      U(3) = 0.0D0
C
      RETURN
      END
```

### 2. 扫频振动（频率线性增加）

```fortran
      SUBROUTINE DISP(U,KSTEP,KINC,TIME,NOEL,NPT,COORDS,JLTYP,
     1 SNAME)
C
C  扫频振动位移 - 频率随时间线性变化
C
      INCLUDE 'ABA_PARAM.INC'
C
      DIMENSION U(3), TIME(2), COORDS(3)
      CHARACTER*80 SNAME
C
      REAL*8 AMP, F0, F1, T_TOTAL, T, PI
      REAL*8 F_INST, PHASE
      PARAMETER(PI=3.141592653589793D0)
C
C-----------------------------------------------------------------------
C  扫频参数
C-----------------------------------------------------------------------
      AMP     = 1.0D-3     ! 振幅 (m)
      F0      = 5.0D0      ! 起始频率 (Hz)
      F1      = 100.0D0    ! 终止频率 (Hz)
      T_TOTAL = 10.0D0     ! 总扫频时间 (s)
C
C-----------------------------------------------------------------------
C  计算瞬时频率（线性扫频）
C-----------------------------------------------------------------------
      T = TIME(2)
      IF (T .GT. T_TOTAL) THEN
        F_INST = F1
      ELSE
        F_INST = F0 + (F1 - F0) * T / T_TOTAL
      END IF
C
C-----------------------------------------------------------------------
C  计算相位（频率的积分）
C  线性扫频：φ(t) = 2π(f₀t + (f₁-f₀)t²/(2T))
C-----------------------------------------------------------------------
      PHASE = 2.0D0*PI*(F0*T + (F1-F0)*T*T/(2.0D0*T_TOTAL))
C
C-----------------------------------------------------------------------
C  计算位移
C-----------------------------------------------------------------------
      U(1) = AMP * SIN(PHASE)
      U(2) = 0.0D0
      U(3) = 0.0D0
C
      RETURN
      END
```

### 3. 空间分布位移（弯曲变形）

```fortran
      SUBROUTINE DISP(U,KSTEP,KINC,TIME,NOEL,NPT,COORDS,JLTYP,
     1 SNAME)
C
C  空间分布位移 - 模拟梁的弯曲变形
C  位移沿长度方向呈三次多项式分布
C
      INCLUDE 'ABA_PARAM.INC'
C
      DIMENSION U(3), TIME(2), COORDS(3)
      CHARACTER*80 SNAME
C
      REAL*8 MAX_DISP, L, X, RAMP, T
      REAL*8 SHAPE_FUNC
C
C-----------------------------------------------------------------------
C  参数
C-----------------------------------------------------------------------
      MAX_DISP = 0.01D0    ! 最大位移 (m)
      L        = 1.0D0     ! 梁长度 (m)
      T = TIME(2)
C
C-----------------------------------------------------------------------
C  时间斜坡函数
C-----------------------------------------------------------------------
      IF (T .LT. 1.0D0) THEN
        RAMP = T
      ELSE
        RAMP = 1.0D0
      END IF
C
C-----------------------------------------------------------------------
C  获取节点X坐标（假设梁沿X方向）
C-----------------------------------------------------------------------
      X = COORDS(1)
C
C-----------------------------------------------------------------------
C  计算形状函数（简支梁一阶弯曲模态）
C  w(x) = sin(πx/L)
C-----------------------------------------------------------------------
      IF (X .GE. 0.0D0 .AND. X .LE. L) THEN
        SHAPE_FUNC = SIN(3.141592653589793D0 * X / L)
      ELSE
        SHAPE_FUNC = 0.0D0
      END IF
C
C-----------------------------------------------------------------------
C  Z方向位移（横向弯曲）
C-----------------------------------------------------------------------
      U(1) = 0.0D0
      U(2) = 0.0D0
      U(3) = MAX_DISP * RAMP * SHAPE_FUNC
C
      RETURN
      END
```

### 4. 旋转运动（刚体转动）

```fortran
      SUBROUTINE DISP(U,KSTEP,KINC,TIME,NOEL,NPT,COORDS,JLTYP,
     1 SNAME)
C
C  刚体旋转位移 - 绕Z轴旋转
C  适用于旋转部件的位移控制
C
      INCLUDE 'ABA_PARAM.INC'
C
      DIMENSION U(3), TIME(2), COORDS(3)
      CHARACTER*80 SNAME
C
      REAL*8 X0, Y0, OMEGA, T, THETA
      REAL*8 X, Y, DX, DY
      REAL*8 X_NEW, Y_NEW
C
C-----------------------------------------------------------------------
C  旋转参数
C-----------------------------------------------------------------------
      X0    = 0.0D0        ! 旋转中心X坐标
      Y0    = 0.0D0        ! 旋转中心Y坐标
      OMEGA = 1.0D0        ! 角速度 (rad/s)
C
C-----------------------------------------------------------------------
C  计算当前转角
C-----------------------------------------------------------------------
      T = TIME(2)
      THETA = OMEGA * T
C
C-----------------------------------------------------------------------
C  获取节点相对于旋转中心的坐标
C-----------------------------------------------------------------------
      X = COORDS(1) - X0
      Y = COORDS(2) - Y0
C
C-----------------------------------------------------------------------
C  计算旋转后的位置
C-----------------------------------------------------------------------
      X_NEW = X * COS(THETA) - Y * SIN(THETA)
      Y_NEW = X * SIN(THETA) + Y * COS(THETA)
C
C-----------------------------------------------------------------------
C  计算位移增量
C-----------------------------------------------------------------------
      DX = X_NEW - X
      DY = Y_NEW - Y
C
      U(1) = DX
      U(2) = DY
      U(3) = 0.0D0
C
      RETURN
      END
```

## 与VDISP的区别

| 特性 | DISP (Standard) | VDISP (Explicit) |
|------|-----------------|------------------|
| 输出 | 总位移 | 总位移 |
| 处理方式 | 每个节点 | 块处理 |
| 典型应用 | 低频振动 | 冲击、瞬态 |

## 注意事项

1. **位移兼容性**：多节点位移应协调避免撕裂
2. **时间连续性**：位移函数应对时间连续可导
3. **速度/加速度**：Abaqus会自动计算速度和加速度
4. **大位移**：几何非线性分析中，位移可能累积
